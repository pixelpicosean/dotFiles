# Dot files

## Dependencies (macOS)

- [brew](http://brew.sh/)
- [nvm](https://github.com/creationix/nvm)

## Dependencies (Windows)

- [Git for Windows]
- [Alacritty]

## Vim setup

1. Install [vim-plug](https://github.com/junegunn/vim-plug)

2. Run the following command to install packages:

`:PlugInstall`
